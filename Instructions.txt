HOW TO:



1) QUIT: TYPE Q OR QUIT

2) REGISTER A NEW USER: TYPE REGUSER AND PROVIDE WHATS MENTIONED

3) LIST ALL REVIEWS: TYPE REVIEWS

4) LIST ALL USERS: TYPE USERS

5) ADD A REVIEW: TYPE ADDREVIEW AND PROVIDE WHATS MENTIONED

6) CHECK A REVIEW: TYPE CHECKREVIEW AND PROVIDE WHATS MENTIONED

7) CHECK A USER: TYPE CHECKUSER AND PROVIDE WHATS MENTIONED

8) SORT BY MOVIE STARS: TYPE SORTBYSTARS

9) SORT BY MOVIE YEAR: TYPE SORTBYDATE

10) SORT BY MOVIE NAME: TYPE SORTBYMOVIE

11) SORT BY THE PERSON WITH THE MOST REVIEWS: TYPE SORTBYNUMB

12) SAVE: TYPE SAVE
